- # Uit readme.txt
  This directory contains the Vtrin ClickOnce deployment files for version 4.50.4874.36859.
  To start Vtrin software from client computer use URL:
  <http://DESSRVHIS1/Vtrin>
  
  When Vtrin starts, it asks to Log On to Database. To access this database, enter the location:
  tcp://DESSRVHIS1/DESSRVHIS1-RTDB
  and Windows username and password that have access to the database.
  The username and password can be left empty if you have logged on with an account that has access to the database.
  
  To provide the connection string to log on to Vtrin automatically use URL:
  <http://DESSRVHIS1/Vtrin?dsn=tcp://DESSRVHIS1/DESSRVHIS1-RTDB&user=username&pass=password>
