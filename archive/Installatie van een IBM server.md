# Installatie van een IBM server

De server moet altijd worden voorzien van de laatste (firmware) drivers. Anders biedt IBM geen ondersteuning.

- Download de laatste versie van ServerGuide (versie9.30 op [[journals/older/2013/2013-01-07]])
- Deze branden op een DVD. Niet op een DVD branden door drivers eruit te slopen. Dit geeft fouten tijdens installatie.
  
  Voor de laatste (firmware)drivers 'bomc' (Bootable Media Creator) downloaden (versie 9.30 op [[journals/older/2013/2013-01-07]])
- Exe uitvoeren
  
  Of installeer IBM UpdateXpress System Pack Installer
