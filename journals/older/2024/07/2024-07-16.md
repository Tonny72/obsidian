## Created Today

```dataview
TABLE string(split(string(file.ctime), " - ")[0]) as "Created", file.mtime as "Last Modified"
WHERE contains(file.path, "2024-07-16")
  OR (file.cday = date("2024-07-16") AND !regexmatch("^\d{4}-\d{2}-\d{2}", file.name))
SORT file.name ASC
```

## Last Modified Today

```dataview
TABLE string(split(string(file.mtime), " - ")[0]) as "Last Modified"
WHERE !contains(file.path, "2024-07-16")
  AND !contains(file.path, "2024-07-16 Daily Note")
  AND file.mday = date("2024-07-16")
SORT file.mtime DESC
```
