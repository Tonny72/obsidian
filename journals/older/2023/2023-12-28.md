# 2023-12-28

- [[Power Bi]] is the way to go for reporting.
	- Build a dedicated [[Parquet]]-file for every report.
- Toml template for config
  ```toml
   -[DES_Energy]
   name = "DES_Energy"
   description = "energie van Dessel"
   period = "1m"
   destinations = ["csv, sql"]
   [DES_Energy.sql]
   name = "sql_table_name"
   [DES_Energy.pq]
   name = "sql_table_name"
   
   [DES_Energy.columns]
   Booster_Rauw_Power = {path = "opc.Applications.HSK15_1.HSK151_Energy.SentronPacs.Booster_Rauw_PM1_:IaPar.Total_Active_Power,1m", period = "1m"}
   Vos2_power = "opc.BOOSTER_RAUW_PM_power.1m"
  ```
