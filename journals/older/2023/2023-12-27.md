- #multi-threading met ThreadPoolExecutor #Python
	- ```python
	  def main():
	      tasks = []
	      executor = ThreadPoolExecutor()
	      for i in range(20):
	          print("create task", i)
	          t = executor.submit(work,i)
	          tasks.append(t)
	      wait(tasks, return_when=ALL_COMPLETED)
	      for t in tasks:
	          print("result work:", t.result())
	      pass
	  ```
- #asyncio #multi-threading #Python
	- ```python
	  async def main():
	      tasks = []
	      for i in range(10):
	          print("create task", i)
	          t = asyncio.create_task(work(i))
	          await asyncio.sleep(random.random())
	          tasks.append(t)
	      done, pending = await asyncio.wait(tasks)
	  ```
