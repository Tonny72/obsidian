# Gebruik Compact Control Builder

Voor het project K287 is gebruik gemaakt van Compact Control Builder.

- # Instellen Compact Control builder voor gebruik met OneDrive
  Na installatie moeten de nodige directories worden ingesteld
  
  Starting the Setup Wizard for Compact Control Builder
  From ABB Start Menu select ABB Compact Product Suite \> Utilities \> **Setup Wizard Control Builder M**.
- ## Working Folder
  Laat deze default staan. Anders komen logfiles ook in deze directory en komen er synchronisatie problemen
- ## Project Folder
  D:\OneDrive - Sibelco\Toepassingen\Compact Control Builder\Projects
- # Backup
  Na elke definitieve versie:
  
  Klik Tools - Maintenace - Project Backup
- # Restore
  Klik Tools - Maintenance - Restore
  
  Doe een REFRESH PROJECT na een restore
- # Versie beheer bibliotheken
  Goed oppassen bij aanpassen bibliotheken. Er is geen versie beheer en bij backup / restore van bibliotheken kunnen nieuwere bibliotheken worden overschreven.
  Eventueel rechtsklik op een bibliotheek en save as. De nieuwe kopie kan naast de originele worden geopend.
- # OPPASSEN !!
  Zet een bibliotheek nooit op Release. In Compact Builder kan deze nooit meer worden aangepast.
  Uit de help
- Released  
  Intended to be used when a library is ready to be shipped to customers; the library is read-only and can never be changed again. (It is recommended to [make a spare copy](mk:@MSITStore:C:\Program%20Files%20(x86)\ABB%20Industrial%20IT\Engineer%20IT\Compact%20Control%20Builder%20AC%20800M\Help\ProjExp2.chm::/PExp0300.htm) of the library before setting state to Released.)
