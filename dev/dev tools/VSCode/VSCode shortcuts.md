#### Tab management

| VSCode              | function                               |
| ------------------- | -------------------------------------- |
| Ctrl w              | Close current tab                      |
| Ctrl Shift T        | Undo close tab                         |
| Ctrl Alt Left/Right | Move the current tab to the split pane |
