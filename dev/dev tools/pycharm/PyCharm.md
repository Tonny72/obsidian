## Shortcut keys

| ShortCut                     | Actions                          |
| ---------------------------- | -------------------------------- |
| Double shift                 | Search everything                |
| Double ctrl                  | Run anything                     |
| Alt Enter                    | Show Context items               |
| F2                           | Navigate between code issues     |
| ctrl E                       | Open recent files                |
| ctrl / *(/ from num keypad)* | Add/Remove line or block comment |
| alt F7                       | find usages                      |
| ctrl shift back              | Last edit location               |
| Ctrl Alt Left/Right          | Navigate back / forward          |
| alt 1                        | toggle explorer                  |
| ctrl alt s                   | open settings                    |
| ctrl shift a                 | command palette                  |
| crtl shift F8                | View breakpoints                 |
| ctrl F8                      | toggle breakpoint                |
| alt 5                        | toggle debug console             |

## Exceptions
- View breakpoints (ctrl shift F8)
- Python Exception Breakpoint
	- Any exception
	- Activation policy
		- On termination
		- Ignore library files
