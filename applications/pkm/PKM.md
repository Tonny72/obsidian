## PKM apps
### [[applications/Obsidian/Obsidian]]
- Standaard markdown wat het samenwerken met andere programma's eenvoudig maakt.
- Gebruik van FrontMatter om een page properties te geven.
### [[applications/pkm/LogSeq]]
- Open Source
- Outliner
- Geen standaard markdown
- LogSeq verknoeit toch wel echt de layout van de oude OneNote's
### Notable (reviewed [[2024-06-06]])
	- Zeer summier
	- geen plugins
	- geen draw functie (geen mermaid, geen plantuml)
### [[applications/pkm/Notion]]
	- Database first app. Meaning that everything is stored in a database style.
	- Can't add pictures in a table
	- 9.50€ / maand
### [[applications/pkm/anyType]] (reviewed [[2024-06-06]])
	- Data is stored in a closed proprietary format.
	- heeft geen plugins
	- heeft geen ondersteuning voor plantuml. volgens forums wel voor mermaid, maar krijg het niet gevonden
### [[dev/dev tools/Microsoft Loop]]
	- still underdeveloped.
### [[applications/pkm/Tana]]
	- https://app.tana.inc/
	- 8€ / maand
### [[applications/pkm/anyType]]
### [[applications/pkm/SiYuan]] (reviewed [[2024-06-06]])

	- no plugin for mermaid or plantuml
