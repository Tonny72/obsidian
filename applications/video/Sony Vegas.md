# Disable Resample
Om een stabiel beeld tijdens beweging te krijgen moet voor elke clip in het project de resample worden disabled.
  
Klik op de selection edit tool
![image1](image1-13.png)
  
Selecteer alle video clips in het project.
  
Klik Edit - Switches - Disable Resample
