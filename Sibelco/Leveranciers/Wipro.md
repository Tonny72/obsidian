# 28/11/2019

Het verhaal met [[Wipro]] is hopeloos.

Vorige week aanvraag ingediend voor een kiosk-pc voor K287. Deze pc is vrij snel geleverd. Maar je moet extra ticket inleggen voor een user. Een extra ticket inleggen om admin rechten te verkrijgen. Deze zijn mij geweigerd. Ik kan dus niks installeren.

Ik heb hiervoor een mail naar Herwig en Bart gestuurd in de hoop dat zij mij kunnen helpen.

Stijn komt mij vertellen dat de CPU load van de MES server verhoogd is, en soms 100%. Er is een nieuwe folder op MES server INVENTORYCLIENT.

Nog deze week o365smtp.sibelco.com is verdwenen. Gevolg dat er geen mails vanuit ABB vertrokken en dat de laag waterstand van de vijver bijna fataal was voor de ovens.

Ik heb met Jef een maand geleden afgesproken om de VMWare van Heerlen in Vcenter te integreren. Zodat er tenminste automatisch een backup kan worden genomen. Nog altijd niet in orde.

Twee pc's voor MHZ CONX en Frothmaster, na een jaar nog altijd niet in orde.

Deze week moest ik de IP telefoon van natzand gaan nakijken. Deze is gewoon verdwenen. Ticket stond een jaar open.
